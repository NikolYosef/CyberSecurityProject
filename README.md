# Cyber Security Project

## Overview
This project is a full-stack cyber security system that includes:

- TLS-based secure communication
- Certificate Authority (CA) authentication
- Anomaly detection mechanism
- Admin panel for equipment and user management
- FastAPI backend
- React frontend

---

## Architecture

Frontend: React  
Backend: FastAPI  
Database: PostgreSQL  
Security: TLS, RSA, AES encryption  

---

## Features

- Secure login system
- Certificate verification
- Encrypted communication
- Admin dashboard
- Anomaly detection alerts
- Equipment management
